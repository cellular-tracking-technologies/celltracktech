# celltracktech

R package for working with Cellular Tracking Technologies Sensor Station and Node data.

**Note**: this package is NOT for analyzing Satellite or GSM tracking devices. Only digital radio and blu series tags.

To install this package, please run:

```         
library(devtools)
install_github('cellular-tracking-technologies/celltracktech')
```

## Tutorial

For a tutorial on how to use this package to analyze your Sensor Station and Node data, click the link below. You will need the API key for the "Meadows V2" project for this tutorial. You can request that API key [here](https://celltracktech.com/pages/csd-radio-api-key-request).

[celltracktech Tutorial](https://cellular-tracking-technologies.github.io/)
